LƯU Ý:
Đây là tính năng mới, không được reset website, database hoặc bất kỳ dữ liệu nào.

==================================================
MỤC TIÊU
==================================================

Tạo một Robot Auto Sync chạy 24/7.

Robot sẽ:

1. Đăng nhập website bán hàng.
2. Truy cập trang Đơn hàng.
3. Tự bấm nút "Tải xuống".
4. Chờ file XLSX tải xong.
5. Đọc dữ liệu.
6. Chỉ import đơn mới.
7. Update đơn thay đổi.
8. Xóa file tạm.
9. Lặp lại theo chu kỳ.
